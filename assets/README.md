# Assets Module

## Descrizione

Contiene le risorse statiche dell'applicazione: CSS, JS, immagini, upload utenti.

## Sotto-cartelle

- `css/` - Fogli di stile, layout e teming
- `js/` - Script lato client, inclusi i script di calendario
  - `calendario_modifica.js` - Logica di editing del calendario in UI
- `images/` - Icone e asset statici
- `uploads/` - File caricati dagli utenti

## Dipendenze

- Librerie client come Tailwind, Font Awesome, SortableJS
